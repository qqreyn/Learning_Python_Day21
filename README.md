# Learning_Python_Day21

Today's topic is the weekly project!

**Topic:** Weekly Project – API Automation Tool  

**Goal:** Build a complete automation tool that fetches, processes, and reports on API data

**Project:**
Complete Automation Tool:

- Fetch data from a chosen API (news, weather, stocks, GitHub, etc.)  
- Process and clean the retrieved data  
- Perform analysis or aggregation on the data  
- Generate a formatted report (display in console and save to file)  
- Save raw data for historical tracking  
- Include proper error handling and logging  
- Create `requirements.txt` and a `README.md` file for the project  
- Optional: Add scheduling capability to run the script automatically
